Regression Plane
You can manipulate the 3D model by changing the 3 input values of H, A, L.
H(Height) controls the starting height of the regression plane.
A(Ascension) controls the rate of increase in the z-axis.
L(Lean) controls the rotation along the z-axis.
Data Points
You can freely add or remove existing data points in the editing panel.
Enter the x, y, z coordinates and press the green "+" to add a datapoint.
The datapoint will then appear in the list and in the 3D model.
To remove a data point simply press the red "x" next to it.
To the right of the coordinates each data points residuals are displayed.
Res is the distance from the z-axis of the data point to the z-axis of the regression plane in the using the same x, y coordinates.
Res2 is the residual value squared.
At the bottom of the list the sum of all Res2 values are displayed.
The lower the number the closer the regression plane accuratly displays the correlation between the x, y, z axis.
